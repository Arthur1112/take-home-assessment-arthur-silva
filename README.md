## Getting Started

```bash
npm install
```

## Scripts

| Command             | Description                   |
| ------------------- | ----------------------------- |
| `npm run dev`       | Start the Vite dev server     |
| `npm run test`      | Run tests with Vitest         |
| `npm run storybook` | Launch Storybook on port 6006 |
